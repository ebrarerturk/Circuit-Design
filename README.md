# Active Low-Pass Filter
This is a small LTspice project I made while learning analog electronics and circuit simulation.
The circuit is a first order active low-pass filter using an RC network and an operational amplifier in a non-inverting configuration.

# What I practiced
- LTspice schematic design
- Op-amp feedback
- AC sweep analysis
- Bode plot interpretation
- Frequency response basics
