# Op-Amp Audio Preamp

This project was created in LTSpice while learning analog electronics and operational amplifier circuits.

The circuit is a simple non-inverting op-amp audio preamplifier with RC filtering components.  
The main goal was to understand how signal amplification, feedback resistors, and basic filtering work in analog circuit design.

In this project I practiced:

- Op-amp power connections
- Non-inverting amplifier configuration
- Feedback resistor design
- RC filter implementation
- AC sweep simulation
- Frequency response analysis in LTSpice
